# 🌤️ WeatherHub - Premium Weather Application

A modern, fully-featured weather application built with HTML, CSS, and JavaScript. Get real-time weather data, forecasts, and alerts with a beautiful, responsive interface.

## ✨ Features

### 🎯 Core Features
- **Real-time Weather Data** - Get current weather conditions instantly
- **Search Functionality** - Search any city in the world
- **Current Location** - 📍 Get weather for your current location using geolocation
- **7-Day Forecast** - View detailed weather predictions
- **Hourly Forecast** - Hour-by-hour weather breakdown

### 🎨 User Interface
- **Dark/Light Mode** - Toggle between themes with smooth transitions
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Dynamic Backgrounds** - Background changes based on weather conditions
  - ☀️ Sunny → Orange gradient
  - ☁️ Cloudy → Grey-blue gradient
  - 🌧️ Rainy → Dark blue gradient
  - 🌙 Night → Dark navy gradient
- **Animated Weather Icons** - Lottie animations for beautiful weather representation
- **Smooth Animations** - CSS animations for seamless page transitions

### 📊 Weather Details
- Temperature (High/Low)
- "Feels Like" temperature with comfort indicator
- Humidity percentage
- Wind speed & direction (compass-based)
- Atmospheric pressure
- Visibility distance
- Sunrise & Sunset times

### 🔔 Notifications & Alerts
- Heat warnings (Temperature > 40°C)
- Rain alerts with umbrella reminder
- Professional alert cards
- No severe weather alerts display

### 🌈 Advanced Features
- **UV Index** - Color-coded scale (Green → Safe to Purple → Extreme)
- **Air Quality** - AQI with PM2.5 and PM10 measurements
- **Precipitation** - Rain chance percentage with expected timing
- **Search Suggestions** - City autocomplete as you type
- **Recent Searches** - Quick access to previously searched cities
- **Favorite Cities** - Save and manage your favorite locations
- **Keyboard Shortcuts**
  - `Ctrl + K` - Open search
  - `Esc` - Close popups
  - `Enter` - Search city

### ⚙️ Settings
- Temperature Units (Celsius/Fahrenheit)
- Wind Speed Units (m/s, km/h, mph)
- Theme Selection (Auto, Light, Dark)
- Notifications Toggle

### 🔄 Data Management
- LocalStorage for saving preferences
- Favorite cities persistence
- Recent searches history
- Settings synchronization

### 📱 Responsive Design
- **Desktop** - Full sidebar + info panel layout
- **Tablet** - Optimized grid layout
- **Mobile** - Bottom navigation menu
- Scrollable content areas
- Touch-friendly buttons
